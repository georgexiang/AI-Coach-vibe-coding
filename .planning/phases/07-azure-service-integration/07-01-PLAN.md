1. Create `backend/app/services/config_service.py` with async functions:
   - `async def get_all_configs(db: AsyncSession) -> list[ServiceConfigResponse]`:
     Query all ServiceConfig rows, build ServiceConfigResponse for each with masked_key = `"****" + decrypt_value(row.api_key_encrypted)[-4:]` if key exists, else "".
   - `async def get_config(db: AsyncSession, service_name: str) -> ServiceConfig | None`:
     Query by service_name, return the row or None.
   - `async def upsert_config(db: AsyncSession, service_name: str, display_name: str, update: ServiceConfigUpdate, updated_by: str) -> ServiceConfig`:
     - Query existing by service_name.
     - If exists: update fields (endpoint, model_or_deployment, region). If update.api_key is non-empty, encrypt and set api_key_encrypted. Set is_active=True, updated_by.
     - If not: create new ServiceConfig with all fields, encrypt api_key.
     - `db.flush()` after changes (per project convention -- let session middleware commit).
     - Return the ServiceConfig.
   - `async def get_decrypted_key(db: AsyncSession, service_name: str) -> str`:
     Get config, decrypt api_key_encrypted, return plaintext. Return "" if not found.
