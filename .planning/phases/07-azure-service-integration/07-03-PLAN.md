3. Update `backend/app/main.py` lifespan -- add DB config loading after mock registration:
   Add async function `_load_configs_from_db()` that:
   - Creates a new async session from `AsyncSessionLocal` (import from database.py).
   - Queries all ServiceConfig rows where is_active=True.
   - For each active config, decrypts api_key and calls `_register_adapter_from_config`.
   - Import the register helper from azure_config.py: `from app.api.azure_config import register_adapter_from_config`.
   Actually, to avoid circular imports, define the registration logic as a standalone function in a new location or inline in main.py lifespan. The cleanest approach: move `_register_adapter_from_config` logic inline in main.py after the mock registration block. Use `from app.utils.encryption import decrypt_value` and `from app.models.service_config import ServiceConfig`.

   In the lifespan, after mock adapter registration and before `yield`:
   ```python
   # Load active configs from DB and register real adapters
   from app.database import AsyncSessionLocal
   from app.models.service_config import ServiceConfig
   from app.utils.encryption import decrypt_value
   async with AsyncSessionLocal() as session:
       from sqlalchemy import select
       result = await session.execute(select(ServiceConfig).where(ServiceConfig.is_active == True))
       configs = result.scalars().all()
       for cfg in configs:
           api_key = decrypt_value(cfg.api_key_encrypted)
           if cfg.service_name == "azure_openai" and api_key:
               from app.services.agents.adapters.azure_openai import AzureOpenAIAdapter
               registry.register("llm", AzureOpenAIAdapter(
                   endpoint=cfg.endpoint, api_key=api_key, deployment=cfg.model_or_deployment
               ))
               settings.default_llm_provider = "azure_openai"
           elif cfg.service_name == "azure_speech_stt" and api_key:
               from app.services.agents.stt.azure import AzureSTTAdapter
               registry.register("stt", AzureSTTAdapter(api_key, cfg.region))
               settings.default_stt_provider = "azure"
           elif cfg.service_name == "azure_speech_tts" and api_key:
               from app.services.agents.tts.azure import AzureTTSAdapter
               registry.register("tts", AzureTTSAdapter(api_key, cfg.region))
               settings.default_tts_provider = "azure"
           elif cfg.service_name == "azure_avatar" and api_key:
               from app.services.agents.avatar.azure import AzureAvatarAdapter
               registry.register("avatar", AzureAvatarAdapter(cfg.endpoint, api_key))
   ```
