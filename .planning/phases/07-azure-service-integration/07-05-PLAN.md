class CoachEventType(StrEnum):
    TEXT = "text"
    AUDIO = "audio"
    SCORE = "score"
    SUGGESTION = "suggestion"

@dataclass
class CoachEvent:

    content: str
    metadata: dict | None = None

@dataclass
class CoachRequest:
    session_id: str
    message: str

    scenario_context: str = ""

class BaseCoachingAdapter(ABC):
    name: str = ""
    @abstractmethod
    async def execute(self, request: CoachRequest) -> AsyncIterator[CoachEvent]: ...
    @abstractmethod
    async def is_available(self) -> bool: ...

   - `name = "azure_content"`
   - `__init__(self, endpoint: str, api_key: str)`: Store `self._endpoint = endpoint.rstrip("/")`, `self._api_key = api_key`
   - `async def execute(self, request: CoachRequest) -> AsyncIterator[CoachEvent]`:
     - Import httpx at module top (not conditional -- httpx is always installed)
     - POST `{self._endpoint}/contentunderstanding/analyzers/prebuilt-invoice:analyze?api-version=2025-11-01`
     - Headers: `{"Ocp-Apim-Subscription-Key": self._api_key, "Content-Type": "application/json"}`
     - Body: `{"url": request.message}`
     - If 202 response: extract `Operation-Location` header, then poll with bounded retry:
       ```python
       MAX_POLL_ATTEMPTS = 30
       POLL_INTERVAL_SECONDS = 2.0
       for attempt in range(MAX_POLL_ATTEMPTS):
           await asyncio.sleep(POLL_INTERVAL_SECONDS)
           poll_response = await client.get(
               operation_url,
               headers={"Ocp-Apim-Subscription-Key": self._api_key},
           )
           poll_data = poll_response.json()
           if poll_data.get("status") == "Succeeded":
               break
           if poll_data.get("status") in ("Failed", "Cancelled"):
               yield CoachEvent(type=CoachEventType.ERROR, content=f"Analysis {poll_data['status']}: {poll_data.get('error', {}).get('message', 'Unknown error')}")
               yield CoachEvent(type=CoachEventType.DONE, content="")
               return
       else:
           yield CoachEvent(type=CoachEventType.ERROR, content="Analysis timed out after 60 seconds")
           yield CoachEvent(type=CoachEventType.DONE, content="")
           return
       ```
     - On success: yield `CoachEvent(type=CoachEventType.TEXT, content=json.dumps(poll_data.get("result", {})))` then DONE
     - On non-202 initial response: yield ERROR with `f"HTTP {response.status_code}: {response.text[:200]}"` then DONE
     - Wrap entire body in try/except: on any exception yield ERROR + DONE (matching azure_openai.py pattern)
   - `async def is_available(self) -> bool`: `return bool(self._endpoint and self._api_key)`
   - Addresses review concern: "Content Understanding polling lacks explicit max wait/backoff/cancel policy" by adding MAX_POLL_ATTEMPTS=30 and POLL_INTERVAL_SECONDS=2.0 (60s max)

   - Else: return model or "gpt-4o-realtime-preview"
   - Note: Model mode stores just the plain model string (backward compatible), Agent mode stores JSON

Function `def get_region_capabilities(region: str) -> dict`:
- Normalize: `region_lower = region.lower().strip()`
- Build result dict with one entry per ALL_SERVICE_NAMES item:
  - For services in UNRESTRICTED_SERVICES: `{"available": True, "note": ""}`
  - For `azure_avatar`:
    - available: `region_lower in AVATAR_REGIONS`
    - note: `"Available in: eastus2, northeurope, southcentralus, southeastasia, swedencentral, westeurope, westus2"` if NOT available, else `""`
  - For `azure_voice_live`:
    - available: `region_lower in VOICE_LIVE_REGIONS`
    - If available AND in VOICE_LIVE_AGENT_REGIONS: note = `"Agent and Model modes available"`
    - If available AND NOT in VOICE_LIVE_AGENT_REGIONS: note = `"Model mode only; Agent mode not available in this region"`
    - If NOT available: note = `"Not available in this region"`
- Return: `{"region": region, "services": result_dict}`
