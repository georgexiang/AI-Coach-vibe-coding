1. **Create `backend/app/services/agent_sync_service.py`** with the following functions:

```python
"""Agent sync service: bidirectional sync between HCP profiles and AI Foundry Agents.

Creates, updates, and deletes AI Foundry Agents via REST API (/assistants endpoint)
when HCP profiles are created, updated, or deleted. Uses httpx for async HTTP calls
with api-key header auth (matching project convention from connection_tester.py).
"""

import json
import httpx
from sqlalchemy.ext.asyncio import AsyncSession
from app.services import config_service
from app.services.agents.adapters.azure_voice_live import parse_voice_live_mode

AI_FOUNDRY_API_VERSION = "2025-05-01"

DEFAULT_AGENT_TEMPLATE = """You are {name}, a {specialty} specialist.

Personality: {personality_type}
Communication Style: {communication_style_desc} (level: {communication_style}/100)
Emotional State: {emotional_state_desc} (level: {emotional_state}/100)

Background:
- Hospital: {hospital}
- Title: {title}
- Expertise: {expertise_areas}
- Prescribing Habits: {prescribing_habits}
- Key Concerns: {concerns}

Common Objections:
{objections}

Topics You Probe About:
{probe_topics}

Stay in character throughout the conversation. Respond as this HCP would in a real face-to-face interaction with a Medical Representative."""
```

Key functions to implement:

a. `build_agent_instructions(profile_data: dict, template: str | None = None) -> str`:
   - Takes profile dict from `HcpProfile.to_prompt_dict()`
   - Converts list fields (expertise_areas, objections, probe_topics) to comma-separated strings
   - Adds computed descriptor fields:
     - `communication_style_desc`: "direct" if value < 50, else "indirect"
     - `emotional_state_desc`: "calm and open" if value < 30, "neutral" if value < 70, else "resistant"
   - Uses Python `str.format_map()` with `defaultdict(lambda: "")` for safe missing-key handling
   - Uses provided `template` or `DEFAULT_AGENT_TEMPLATE`

b. `async def get_project_endpoint(db: AsyncSession) -> tuple[str, str]`:
   - Gets AI Foundry master config endpoint via `config_service.get_effective_endpoint(db, "azure_voice_live")`
   - Gets project_name from voice_live config's model_or_deployment via `parse_voice_live_mode()`
   - Constructs full project endpoint: `{base_endpoint}/api/projects/{project_name}` if base ends with `.azure.com` or similar
   - If base_endpoint already contains `/api/projects/`, use it as-is
   - Returns `(project_endpoint, api_key)`
   - Gets API key via `config_service.get_effective_key(db, "azure_voice_live")`

c. `async def create_agent(db: AsyncSession, name: str, instructions: str, model: str = "gpt-4o") -> dict`:
   - Gets project endpoint and API key via `get_project_endpoint(db)`
   - POST to `{project_endpoint}/assistants?api-version=2025-05-01`
   - Headers: `{"api-key": api_key, "Content-Type": "application/json"}`
   - Body: `{"model": model, "name": name, "instructions": instructions}`
   - Uses `httpx.AsyncClient(timeout=30.0)`
   - Calls `response.raise_for_status()`
   - Returns `response.json()` (contains `id` field = agent_id)

d. `async def update_agent(db: AsyncSession, agent_id: str, name: str, instructions: str) -> dict`:
   - POST to `{project_endpoint}/assistants/{agent_id}?api-version=2025-05-01`
   - Same headers as create
   - Body: `{"name": name, "instructions": instructions}`
   - Returns `response.json()`

e. `async def delete_agent(db: AsyncSession, agent_id: str) -> bool`:
   - DELETE to `{project_endpoint}/assistants/{agent_id}?api-version=2025-05-01`
   - Returns `response.status_code == 200`

f. `async def sync_agent_for_profile(db: AsyncSession, profile, template: str | None = None) -> dict`:
   - High-level helper that creates or updates agent based on whether `profile.agent_id` is set
   - Calls `profile.to_prompt_dict()` to get profile data
   - Calls `build_agent_instructions(profile_data, template)`
   - Gets model from master config's `model_or_deployment` via `config_service.get_master_config(db)`
   - If `profile.agent_id` is truthy: calls `update_agent(db, profile.agent_id, profile.name, instructions)`
   - Else: calls `create_agent(db, profile.name, instructions, model)`
   - Returns the API response dict
