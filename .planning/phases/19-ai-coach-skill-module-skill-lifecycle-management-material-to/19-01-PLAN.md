1. Create `backend/app/services/skill_service.py`. Addresses review concerns: strict transition enforcement, single published version invariant, optimistic locking, complete file security.

   **Module-level constants:**
   ```python
   ALLOWED_RESOURCE_EXTENSIONS = {".pdf", ".docx", ".pptx", ".txt", ".md", ".py", ".json", ".csv", ".xlsx"}
   MAX_FILE_SIZE = 50 * 1024 * 1024  # 50MB per file
   MAX_FILES_PER_UPLOAD = 10
   ```

   **def validate_status_transition(current: str, target: str) -> None:**
   - Import VALID_TRANSITIONS from models.skill
   - If target not in VALID_TRANSITIONS.get(current, set()): raise bad_request(f"Invalid status transition from '{current}' to '{target}'. Allowed: {VALID_TRANSITIONS.get(current, set())}")
   - This is the SOLE transition enforcement point -- no ad-hoc transition logic elsewhere

   **def sanitize_filename(filename: str) -> str:**
   - Strip directory separators, reject `..`, reject absolute paths
   - Use `pathlib.PurePosixPath(filename).name` to extract base filename
   - Reject empty result, reject if contains control characters
   - Return sanitized filename

   **def validate_file_upload(filename: str, file_size: int) -> None:**
   - Check extension against ALLOWED_RESOURCE_EXTENSIONS
   - Check file_size <= MAX_FILE_SIZE
   - Raise bad_request with specific message on failure

   **async def create_skill(db: AsyncSession, data: SkillCreate, user_id: str) -> Skill:**
   - Create Skill record with data fields, created_by=user_id, updated_by=user_id, status="draft"
   - Create initial SkillVersion (version_number=1, content=data.content, metadata_json=data.metadata_json, is_published=False, created_by=user_id)
   - db.flush()
   - Return skill (use selectinload for resources and versions)

   **async def get_skills(db, page=1, page_size=20, status=None, product=None, search=None) -> tuple[list[Skill], int]:**
   - Query with optional filters: status exact match, product exact match, search ILIKE on name+description
   - Order by created_at desc
   - Use .limit(page_size).offset((page - 1) * page_size) for pagination
   - Return (items, total)

   **async def get_published_skills(db, page=1, page_size=20, search=None) -> tuple[list[Skill], int]:**
   - Same as get_skills but hardcoded status="published"

   **async def get_skill(db: AsyncSession, skill_id: str) -> Skill:**
   - Load by id with selectinload(Skill.versions) and selectinload(Skill.resources)
   - Raise not_found("Skill not found") if not exists

   **async def update_skill(db: AsyncSession, skill_id: str, data: SkillUpdate, user_id: str) -> Skill:**
   - Load skill
   - If data.status is set and differs from current: call validate_status_transition(skill.status, data.status)
   - Apply non-None fields from data
   - Set skill.updated_by = user_id
   - db.flush()
   - Return updated skill

   **async def delete_skill(db: AsyncSession, skill_id: str) -> None:**
   - Load skill. Only allow deletion if status in ("draft", "failed")
   - Raise bad_request("Cannot delete skill in '{status}' status. Only draft or failed skills can be deleted.") otherwise
   - Delete associated resources' files from storage
   - db.delete(skill), db.flush()

   **async def publish_skill(db: AsyncSession, skill_id: str, user_id: str) -> Skill:**
   - Load skill
   - Enforce: skill.status must be "review" (use validate_status_transition)
   - Enforce: structure_check_passed must be True, else raise bad_request("L1 structure check must pass before publishing")
   - Enforce: quality_score must not be None and >= 50, else raise bad_request("Quality score must be at least 50 to publish")
   - **Invariant: single published version.** Mark all existing SkillVersions for this skill as is_published=False
   - Create new SkillVersion snapshot: version_number=skill.current_version, content=skill.content, metadata_json=skill.metadata_json, is_published=True, created_by=user_id
   - Set skill.status = "published", skill.updated_by = user_id
   - db.flush()

   **async def create_new_version(db: AsyncSession, skill_id: str, user_id: str) -> Skill:**
   - Load skill, verify status == "published"
   - Increment current_version
   - Set status = "draft"
   - Reset quality fields: structure_check_passed=None, quality_score=None, quality_verdict=None, quality_details="{}", structure_check_details="{}"
   - Set updated_by = user_id
   - db.flush()

   **async def archive_skill(db: AsyncSession, skill_id: str, user_id: str) -> Skill:**
   - Load skill, validate_status_transition(skill.status, "archived")
   - Set status="archived", updated_by=user_id
   - db.flush()

   **async def restore_skill(db: AsyncSession, skill_id: str, user_id: str) -> Skill:**
   - Load skill, validate_status_transition(skill.status, "draft")
   - Set status="draft", updated_by=user_id
   - db.flush()
