   **Query-key factory (module-level constant):**
   ```typescript
   export const skillKeys = {
     all: ["skills"] as const,
     lists: () => [...skillKeys.all, "list"] as const,
     list: (params: Record<string, unknown>) => [...skillKeys.lists(), params] as const,
     published: (params?: Record<string, unknown>) => [...skillKeys.all, "published", params] as const,
     details: () => [...skillKeys.all, "detail"] as const,
     detail: (id: string) => [...skillKeys.details(), id] as const,
     conversion: (id: string) => [...skillKeys.detail(id), "conversion"] as const,
     evaluation: (id: string) => [...skillKeys.detail(id), "evaluation"] as const,
   };
   ```

   **Mutation invalidation matrix:**
   - Create/Delete/Archive/Restore/Publish: invalidate skillKeys.all (all lists + details)
   - Update: invalidate skillKeys.detail(id) + skillKeys.lists()
   - Conversion: invalidate skillKeys.detail(id) + skillKeys.conversion(id)
   - Quality check: invalidate skillKeys.detail(id) + skillKeys.evaluation(id)

   **Hooks (all use skillKeys factory):**
   - useSkills(params?) -- queryKey: skillKeys.list(params)
   - usePublishedSkills(params?) -- queryKey: skillKeys.published(params)
   - useSkill(id) -- queryKey: skillKeys.detail(id), enabled: !!id
   - useCreateSkill(), useUpdateSkill(), useDeleteSkill(), usePublishSkill(), useArchiveSkill(), useRestoreSkill() -- mutations with proper invalidation via skillKeys
   - useConversionStatus(id, enabled) -- queryKey: skillKeys.conversion(id), refetchInterval: 3000 when enabled
   - useStartConversion(), useRetryConversion() -- mutations
   - useCheckStructure(), useEvaluateQuality() -- mutations
   - useSkillEvaluation(id) -- queryKey: skillKeys.evaluation(id)
   - useUploadResources() -- mutation
   - useRegenerateSop() -- mutation that invalidates skillKeys.detail(id)
