   **Security constants (addresses review: ZIP import security gaps):**
   ```python
   import io
   import logging
   import os
   import zipfile
   from pathlib import PurePosixPath

   logger = logging.getLogger(__name__)

   # Hard limits for ZIP import security
   MAX_ZIP_SIZE_BYTES = 100 * 1024 * 1024          # 100MB compressed
   MAX_UNCOMPRESSED_SIZE_BYTES = 100 * 1024 * 1024  # 100MB uncompressed total
   MAX_ZIP_ENTRIES = 500                             # Max files in archive
   MAX_PATH_DEPTH = 5                                # Max directory nesting
   MAX_SINGLE_FILE_SIZE = 50 * 1024 * 1024           # 50MB per file
   ALLOWED_IMPORT_EXTENSIONS = {
       ".md", ".txt", ".pdf", ".docx", ".pptx",
       ".py", ".json", ".yaml", ".yml", ".csv",
       ".png", ".jpg", ".jpeg", ".svg", ".gif",
   }
   ALLOWED_DIRECTORIES = {"references", "scripts", "assets"}
   ```

   **def validate_zip_security(zip_bytes: bytes) -> list[str]:**
   Addresses review: comprehensive ZIP security validation.
   - Check compressed size <= MAX_ZIP_SIZE_BYTES
   - Open as zipfile.ZipFile(io.BytesIO(zip_bytes))
   - Check entry count <= MAX_ZIP_ENTRIES
   - For each entry in zip.infolist():
     a) **Path traversal check:** reject if ".." in entry.filename or entry.filename starts with "/"
     b) **Symlink check:** reject if entry.external_attr indicates symlink (entry.external_attr >> 28 == 0xA)
     c) **Path depth check:** reject if PurePosixPath(entry.filename).parts count > MAX_PATH_DEPTH
     d) **File size check:** reject if entry.file_size > MAX_SINGLE_FILE_SIZE
     e) **Extension check:** reject if file extension not in ALLOWED_IMPORT_EXTENSIONS (skip directories)
     f) **Directory check:** for non-root files, reject if top-level directory not in ALLOWED_DIRECTORIES and file is not SKILL.md
   - Sum all entry.file_size values: reject if total > MAX_UNCOMPRESSED_SIZE_BYTES (zip bomb protection)
   - Return list of validation error strings (empty = valid)

   **async def export_skill_zip(db: AsyncSession, skill_id: str) -> bytes:**
   - Load Skill with selectinload(Skill.resources, Skill.versions)
   - If not found: raise not_found("Skill not found")
   - Create zipfile.ZipFile in memory (io.BytesIO, mode='w', compression=zipfile.ZIP_DEFLATED)
   - Build SKILL.md content:
     a) Reconstruct YAML frontmatter from DB fields: name, description, version (current_version), product, therapeutic_area, compatibility, tags, metadata
     b) Append "---" separator
     c) Append skill.content (Markdown body)
   - Write "SKILL.md" to zip
   - For each SkillResource:
     - Determine path: f"{resource.resource_type}s/{resource.filename}" (references/, scripts/, assets/)
     - Write text_content to zip if available, else write empty placeholder with comment
   - Close zip, return bytes

   **async def import_skill_zip(db: AsyncSession, zip_bytes: bytes, created_by: str = "admin") -> Skill:**
   Addresses review: reject duplicates, validate manifest, scripts as inert artifacts.
   - Run validate_zip_security(zip_bytes): if errors, raise ValidationException with all error strings
   - Open zip
   - **Find and parse SKILL.md:**
     - Look for "SKILL.md" in zip (case-sensitive)
     - If not found: raise bad_request("ZIP must contain a SKILL.md file at root")
     - Read content, split on first "---\n" to separate YAML frontmatter from Markdown body
     - Parse YAML frontmatter (use yaml.safe_load) for: name, description, product, therapeutic_area, compatibility, tags
   - **Conflict check (addresses review: import conflict policy = reject duplicates):**
     - Query existing Skill with same name
     - If found: raise ConflictException("A skill with name '{name}' already exists. Rename or delete the existing skill before importing.")
   - **Create Skill:**
     - Call skill_service.create_skill(db, SkillCreate(name=..., description=..., content=markdown_body, ...))
   - **Import resources:**
     - For each file in zip (excluding SKILL.md):
       - Determine resource_type from directory prefix: "references/" -> "reference", "scripts/" -> "script", "assets/" -> "asset"
       - Read file content as text (with UTF-8 fallback to latin-1)
       - Create SkillResource with text_content, filename, resource_type
       - Scripts are stored as text_content only -- NEVER auto-executed (addresses review: imported scripts inert)
       - Log: logger.info("Imported resource: %s (%s)", filename, resource_type)
   - db.flush()
   - Return skill

   Import yaml at top: `import yaml` (already available via pydantic-settings dependency)

   **POST /import** (per D-27):
   ```python
   @router.post("/import", status_code=201)
   async def import_skill(
       file: UploadFile = File(...),
       db: AsyncSession = Depends(get_db),
   ):
       if not file.filename or not file.filename.endswith(".zip"):
           raise bad_request("File must be a .zip archive")
       zip_bytes = await file.read()
       if len(zip_bytes) > skill_zip_service.MAX_ZIP_SIZE_BYTES:
           raise bad_request(f"ZIP file exceeds maximum size of {skill_zip_service.MAX_ZIP_SIZE_BYTES // (1024*1024)}MB")
       skill = await skill_zip_service.import_skill_zip(db, zip_bytes)
       await db.commit()
       return SkillResponse.model_validate(skill)
   ```
