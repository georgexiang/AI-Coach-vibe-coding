   **@dataclass SkillContent:**
   - name: str, description: str, content: str, version_id: str, token_estimate: int

   **class SkillManager:**

   **@staticmethod
   def from_db_skill(skill: Skill, version: SkillVersion | None = None) -> SkillContent:**
   - Use version.content if version provided (deterministic per pinned version), else skill.content
   - Return SkillContent(name=skill.name, description=skill.description, content=version.content if version else skill.content, version_id=version.id if version else "", token_estimate=len(content) // 4)

   **@staticmethod
   def compose_instructions(base_instructions: str, skills: list[SkillContent], max_tokens: int = 4000) -> str:**
   - Follow reference implementation:
     ```python
     parts = [base_instructions]
     total_tokens = len(base_instructions) // 4
     for skill in skills:
         header = f"== Skill: {skill.name} (v:{skill.version_id[:8]}) =="
         section = f"\n\n{header}\n{skill.description}\n\n{skill.content}"
         parts.append(section)
         total_tokens += skill.token_estimate
     if total_tokens > max_tokens:
         logger.warning("Token estimate (%d) exceeds max_tokens (%d)", total_tokens, max_tokens)
     return "".join(parts)
     ```

   **async def load_skill_for_scenario(db: AsyncSession, scenario_id: str) -> SkillContent | None:**
   - Query Scenario by id
   - If no skill_id: return None
   - Load Skill by skill_id
   - If skill not found or status not in ("published", "archived"): return None
   - Load SkillVersion by scenario.skill_version_id if set (deterministic), else find is_published=True version
   - Return SkillContent using from_db_skill(skill, version)

   **def run_skill_script(script_content: str, timeout: int = 30) -> str:**
   Addresses review: sandboxed script execution with restricted subprocess.
   ```python
   import subprocess
   import tempfile
   import os

   # Security: write to temp file instead of -c flag (avoids shell interpretation)
   with tempfile.NamedTemporaryFile(mode='w', suffix='.py', delete=False, dir='/tmp') as f:
       f.write(script_content)
       script_path = f.name

   try:
       result = subprocess.run(
           ["python3", script_path],
           capture_output=True,
           text=True,
           timeout=timeout,
           shell=False,           # No shell (addresses review: sandbox)
           cwd="/tmp",            # Restricted working directory
           env={                  # Minimal environment (addresses review: resource controls)
               "PATH": "/usr/bin:/usr/local/bin",
               "HOME": "/tmp",
               "LANG": "en_US.UTF-8",
           },
       )
       if result.returncode != 0:
           raise RuntimeError(f"Script failed (exit {result.returncode}): {result.stderr[:500]}")
       return result.stdout
   finally:
       os.unlink(script_path)
   ```
