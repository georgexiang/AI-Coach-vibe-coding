# Plan 18-02: Frontend PDF Preview & File Download

## Goal
Add PDF inline preview and DOCX/XLSX download to the training materials page.

## Context
- Frontend materials page: `frontend/src/pages/admin/training-materials.tsx`
- API client: `frontend/src/api/materials.ts`
- Types: `frontend/src/types/material.ts`
- Version history dialog exists (lines 698-754) with "View Chunks" button but no download/preview
- Backend will provide `download_url` field on each version (from Plan 18-01)

## Tasks

### Task 1: Update TypeScript types
**File:** `frontend/src/types/material.ts`
- Add `download_url: string` to `MaterialVersion` interface
- Remove or deprecate `storage_url` field (replace with `download_url`)

### Task 2: Add download helper to API client
**File:** `frontend/src/api/materials.ts`
- Add `getDownloadUrl(materialId: string, versionId: string, mode?: 'inline' | 'attachment'): string`
  - Returns the full URL including auth token as query param OR uses axios interceptor
- Add `downloadFile(materialId: string, versionId: string): void`
  - Creates a temporary `<a>` element with download attribute and triggers click
  - Uses blob download pattern for authenticated file download

### Task 3: Add PDF preview dialog component
**File:** `frontend/src/components/admin/material-preview-dialog.tsx` (new)
- Dialog with `<iframe>` or `<object>` element for PDF rendering
- Uses download URL with `?mode=inline` as iframe src
- Shows filename in dialog title
- Fallback message if browser doesn't support PDF preview
- Close button

### Task 4: Update version history dialog
**File:** `frontend/src/pages/admin/training-materials.tsx`
- In the version history section, add action buttons per version:
  - PDF files: "Preview" button (opens MaterialPreviewDialog) + "Download" button
  - DOCX/XLSX files: "Download" button only
- Use `content_type` to determine which buttons to show
- Also add download/preview buttons to the material card or list item

### Task 5: Add i18n keys
**Files:**
- `frontend/public/locales/en-US/admin.json`: Add `materials.preview`, `materials.download`, `materials.previewNotSupported`
- `frontend/public/locales/zh-CN/admin.json`: Add Chinese translations

### Task 6: TypeScript build check
```bash
cd frontend && npx tsc -b && npm run build
```
