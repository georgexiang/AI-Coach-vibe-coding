**2. Create `frontend/src/stores/theme-store.ts`** (per D-08, D-09):

Follow the exact `useSyncExternalStore` pattern from `auth-store.ts`. Module-level state with listeners Set.

Types:
```typescript
type ThemeMode = "light" | "dark";
type AccentColor = "blue" | "teal" | "purple" | "rose" | "amber";
interface ThemeState { mode: ThemeMode; accent: AccentColor; }
```

Constants:
- `THEME_STORAGE_KEY = "ai-coach-theme"`
- `ACCENT_STORAGE_KEY = "ai-coach-accent"`
- `ACCENT_COLORS` array of `{ name: AccentColor, color: string, label: string }` with values: blue/#1E40AF/"BeiGene Blue", teal/#0D9488/"Teal", purple/#7C3AED/"Purple", rose/#BE185D/"Rose", amber/#B45309/"Amber"

`applyTheme(mode, accent)` function:
- Toggle `.dark` class on `document.documentElement` based on mode
- Remove all `theme-*` classes from documentElement using regex `/theme-\w+/g`
- If accent is not "blue", add `theme-{accent}` class

Initialization: read from localStorage, call applyTheme on module load.

Export `setThemeMode(mode: ThemeMode)`, `setAccentColor(accent: AccentColor)`, and `useThemeStore()` hook returning `{ mode, accent, setThemeMode, setAccentColor }`.

**3. Add flash prevention script to `frontend/index.html`**:

Insert a synchronous `<script>` tag BEFORE the `<div id="root">`:

```html
<script>
  (function() {
    var mode = localStorage.getItem('ai-coach-theme') || 'light';
    var accent = localStorage.getItem('ai-coach-accent') || 'blue';
    if (mode === 'dark') document.documentElement.classList.add('dark');
    if (accent !== 'blue') document.documentElement.classList.add('theme-' + accent);
  })();
</script>
```
