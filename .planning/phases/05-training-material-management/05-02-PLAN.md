   ```python
   """Material service: CRUD, versioning, text extraction, chunk search."""

   import asyncio

   from sqlalchemy import func, select
   from sqlalchemy.ext.asyncio import AsyncSession
   from sqlalchemy.orm import selectinload

   from app.config import get_settings
   from app.models.material import MaterialChunk, MaterialVersion, TrainingMaterial
   from app.schemas.material import MaterialUpdate
   from app.services.storage import get_storage
   from app.services.text_extractor import extract_text
   from app.utils.exceptions import bad_request, not_found
   ```

   Implement these async functions:

   **`async def upload_material(db, content, filename, content_type, product, name, tags, therapeutic_area, material_id, user_id) -> TrainingMaterial`**
   - If `material_id` is provided: load existing material, get latest version number, increment. Otherwise create new TrainingMaterial record.
   - Store file via `get_storage().save(f"materials/{material.id}/v{version_number}/{filename}", content)`
   - Create MaterialVersion record with version_number, filename, file_size=len(content), content_type, storage_url
   - Run text extraction via `asyncio.to_thread(extract_text, content, content_type)` (CPU-bound, use thread to avoid blocking)
   - For each (page_label, text) tuple returned, create MaterialChunk(version_id=version.id, material_id=material.id, chunk_index=i, content=text, page_label=page_label)
   - Update material.current_version = version_number
   - `await db.flush()` then `await db.refresh(material, attribute_names=["versions"])`
   - Return material

   **`async def get_materials(db, page, page_size, product, search, include_archived) -> tuple[list[TrainingMaterial], int]`**
   - Query TrainingMaterial with optional filters: `product` (exact), `search` (ILIKE on name), `include_archived` (default False, filter `is_archived==False` unless True)
   - Order by created_at desc
   - Return paginated results using offset/limit pattern (like scenario_service)

   **`async def get_material(db, material_id) -> TrainingMaterial`**
   - Load material with selectinload("versions"), raise not_found if not exists

   **`async def update_material(db, material_id, data: MaterialUpdate) -> TrainingMaterial`**
   - Load material, apply non-None fields from data, flush and refresh

   **`async def archive_material(db, material_id) -> TrainingMaterial`**
   - Set is_archived=True, flush and refresh

   **`async def restore_material(db, material_id) -> TrainingMaterial`**
   - Set is_archived=False, flush and refresh

   **`async def get_versions(db, material_id) -> list[MaterialVersion]`**
   - Query versions for material_id ordered by version_number desc

   **`async def get_version_chunks(db, version_id) -> list[MaterialChunk]`**
   - Query chunks for version_id ordered by chunk_index

   **`async def search_chunks(db, product, query, limit) -> list[MaterialChunk]`**
   - Join MaterialChunk -> MaterialVersion -> TrainingMaterial
   - Filter: TrainingMaterial.product == product, TrainingMaterial.is_archived == False
   - Filter: MaterialVersion.is_active == True
   - Filter: MaterialChunk.content.ilike(f"%{query}%")
   - Only return chunks from the latest active version of each material (subquery: max version_number per material_id where is_active=True)
   - Limit results (default 10)
   - Return list of chunks

   **`async def get_material_context(db, product, limit) -> list[str]`**
   - Convenience function: search all chunks for a product (no query filter), return list of chunk.content strings. Used by prompt builder integration.
   - Only get chunks from latest active version of each material for the given product
   - Return up to `limit` chunk content strings (default 20)

   Endpoints (static routes BEFORE parameterized per CLAUDE.md Gotcha #3):

   **`POST /` (status_code=201)** -- Upload material (admin only)
   - Params: `file: UploadFile`, `product: str = Form(...)`, `name: str = Form(...)`, `therapeutic_area: str = Form("")`, `tags: str = Form("")`, `material_id: str | None = Form(None)`
   - Validate extension against ALLOWED_EXTENSIONS, validate file size against MAX_FILE_SIZE
   - Call material_service.upload_material
   - Return MaterialOut

   **`GET /search`** -- Search material chunks (admin only, static route before `/{material_id}`)
   - Params: `product: str = Query(...)`, `query: str = Query("")`, `limit: int = Query(10)`
   - Call material_service.search_chunks
   - Return list[MaterialChunkOut]

   **`GET /`** -- List materials (admin only)
   - Params: page, page_size, product (optional), search (optional), include_archived (bool, default False)
   - Return PaginatedResponse[MaterialListOut]

   **`GET /{material_id}`** -- Get material detail (admin only)
   - Return MaterialOut (with versions)

   **`PUT /{material_id}`** -- Update material metadata (admin only)
   - Body: MaterialUpdate JSON
   - Return MaterialOut
