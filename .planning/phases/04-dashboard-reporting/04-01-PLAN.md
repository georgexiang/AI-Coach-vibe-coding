   ```python
   """Analytics response schemas for dashboard and reporting endpoints."""
   from datetime import datetime
   from pydantic import BaseModel, ConfigDict

   class UserDashboardStats(BaseModel):
       """All four stat card values for user dashboard."""
       total_sessions: int
       avg_score: float
       this_week: int
       improvement: float | None  # latest score minus previous, None if < 2 sessions
       model_config = ConfigDict(from_attributes=True)

   class DimensionTrendPoint(BaseModel):
       """A single data point in a per-dimension time series."""
       session_id: str
       completed_at: datetime | None
       scenario_name: str
       overall_score: float
       dimensions: list[DimensionScore]
       model_config = ConfigDict(from_attributes=True)

   class DimensionScore(BaseModel):
       """Score for one dimension in one session."""
       dimension: str
       score: float
       weight: int
       model_config = ConfigDict(from_attributes=True)

   class BuStats(BaseModel):
       """Per-business-unit aggregated stats."""
       business_unit: str
       session_count: int
       avg_score: float
       user_count: int
       model_config = ConfigDict(from_attributes=True)

   class SkillGapCell(BaseModel):
       """Single cell in the skill gap heatmap matrix."""
       business_unit: str
       dimension: str
       avg_score: float
       model_config = ConfigDict(from_attributes=True)

   class OrgAnalytics(BaseModel):
       """Organization-level analytics for admin dashboard."""
       total_users: int
       active_users: int
       completion_rate: float
       total_sessions: int
       avg_org_score: float
       bu_stats: list[BuStats]
       skill_gaps: list[SkillGapCell]
       model_config = ConfigDict(from_attributes=True)

   class RecommendedScenarioItem(BaseModel):
       """A recommended scenario for the user."""
       scenario_id: str
       scenario_name: str
       product: str
       difficulty: str
       reason: str  # e.g. "Targets your weakest dimension: objection_handling"
       target_dimension: str
       model_config = ConfigDict(from_attributes=True)
   ```

1. Create `backend/app/services/analytics_service.py` with these functions:

   **`get_user_dashboard_stats(db: AsyncSession, user_id: str) -> UserDashboardStats`**
   - Query `func.count()` on CoachingSession where user_id matches and status == "scored" for `total_sessions`
   - Query `func.avg(CoachingSession.overall_score)` for `avg_score`, round to 1 decimal
   - Query `func.count()` on CoachingSession where `created_at >= datetime.now(UTC) - timedelta(days=7)` for `this_week`
   - For `improvement`: query the 2 most recent scored sessions ordered by `completed_at desc`, compute `latest.overall_score - previous.overall_score`. Return None if fewer than 2 scored sessions.
   - Return `UserDashboardStats` schema instance

   **`get_user_dimension_trends(db: AsyncSession, user_id: str, limit: int = 20) -> list[DimensionTrendPoint]`**
   - Load scored sessions ordered by `completed_at desc`, limit to `limit`
   - For each session, eager-load score with details and scenario
   - Return list of `DimensionTrendPoint` with session_id, completed_at, scenario_name, overall_score, and dimensions list
   - This is similar to existing `get_score_history` but returns typed Pydantic schemas

   **`get_org_analytics(db: AsyncSession) -> OrgAnalytics`**
   - Count total users where `role == "user"` for `total_users`
   - Count distinct user_ids in scored sessions for `active_users`
   - Compute `completion_rate = round((active_users / max(total_users, 1)) * 100, 1)`
   - Count total scored sessions for `total_sessions`
   - Compute org-wide `avg_org_score` from all scored sessions
   - Group by `User.business_unit`: join CoachingSession on user_id, where status == "scored", group by business_unit. For each BU: session_count, avg_score, and count distinct user_ids as user_count. Return as `list[BuStats]`
   - Return `OrgAnalytics` schema

   **`get_skill_gap_matrix(db: AsyncSession) -> list[SkillGapCell]`**
   - Join User -> CoachingSession -> SessionScore -> ScoreDetail
   - Filter: CoachingSession.status == "scored", User.role == "user"
   - Group by: User.business_unit, ScoreDetail.dimension
   - Aggregate: func.avg(ScoreDetail.score) as avg_score, round to 1
   - Return list of SkillGapCell(business_unit, dimension, avg_score)

   **`get_recommended_scenarios(db: AsyncSession, user_id: str, limit: int = 3) -> list[RecommendedScenarioItem]`**
   - Step 1: Get user's average score per dimension from last 10 scored sessions
   - Step 2: Find the dimension with the lowest average score (weakest dimension)
   - Step 3: Map dimension name to scenario weight column: {"key_message": Scenario.weight_key_message, "objection_handling": Scenario.weight_objection_handling, "communication": Scenario.weight_communication, "product_knowledge": Scenario.weight_product_knowledge, "scientific_info": Scenario.weight_scientific_info}
   - Step 4: Query active scenarios ordered by the weight column for weakest dimension DESC
   - Step 5: Exclude scenario_ids that user completed in last 5 sessions
   - Step 6: Return top `limit` as RecommendedScenarioItem with reason="Targets your weakest dimension: {dimension_name}"
   - If no scored sessions exist, return empty list

   All functions use `async def` with `AsyncSession`. Import models from `app.models`. Import schemas from `app.schemas.analytics`. Use `from datetime import UTC, datetime, timedelta`. Use `from sqlalchemy import and_, func, select`. Use `from sqlalchemy.orm import selectinload`.
