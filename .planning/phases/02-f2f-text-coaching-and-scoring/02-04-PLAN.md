All session endpoints require `user: User = Depends(get_current_user)` (not admin-only — regular users coach).

Endpoints:
- `POST /` — Create session. Request: `SessionCreate` (contains scenario_id). Response: `SessionResponse`, status 201.

- `GET /` — List user's sessions. Query: page, page_size. Response: paginated `SessionResponse` list. (Returns only sessions belonging to current user.)

- `GET /active` — Get user's currently active session (status in_progress). Response: `SessionResponse` or 404.

- `GET /{session_id}` — Get session detail with messages. Response: `SessionResponse`.

- `POST /{session_id}/message` — Send MR message and stream HCP response via SSE. Request: `SendMessageRequest`. Response: `EventSourceResponse`.

  Implementation per D-01 (SSE streaming) and RESEARCH.md Pattern 1:
  ```python
  async def send_message(session_id: str, request: SendMessageRequest, db: AsyncSession, user: User):
      session = await session_service.get_session(db, session_id, user.id)
      # Reject if session is not "in_progress" or "created" (COACH-09)
      if session.status not in ("created", "in_progress"):
          raise AppException(status_code=409, code="SESSION_CLOSED", message="Session is no longer active")

      # Save MR message (this also transitions created -> in_progress)
      await session_service.save_message(db, session_id, "user", request.message)

      async def event_generator():
          # Build coach request
          adapter = registry.get("llm", settings.default_llm_provider)
          # Build full conversation history (D-11)
          messages = await session_service.get_session_messages(db, session_id)
          # Build HCP system prompt
          hcp_prompt = build_hcp_system_prompt(session.scenario.hcp_profile, session.scenario, json.loads(session.scenario.key_messages))

          coach_request = CoachRequest(
              session_id=session_id,
              message=request.message,
              scenario_context=hcp_prompt,
              hcp_profile=session.scenario.hcp_profile.to_prompt_dict() if hasattr(session.scenario, 'hcp_profile') else None,
              scoring_criteria=session.scenario.get_scoring_weights() if hasattr(session.scenario, 'get_scoring_weights') else None,
          )

          full_response = ""
          async for event in adapter.execute(coach_request):
              if event.type == CoachEventType.TEXT:
                  full_response += event.content
                  yield {"event": "text", "data": event.content}
              elif event.type == CoachEventType.SUGGESTION:
                  yield {"event": "hint", "data": json.dumps({"content": event.content, "metadata": event.metadata})}
              elif event.type == CoachEventType.DONE:
                  # Save complete HCP response
                  await session_service.save_message(db, session_id, "assistant", full_response)
                  # Key message detection (D-03)
                  key_messages_status = await session_service.detect_key_messages(db, session, request.message)
                  yield {"event": "key_messages", "data": json.dumps(key_messages_status)}
                  yield {"event": "done", "data": ""}

      return EventSourceResponse(event_generator())
  ```

- `POST /{session_id}/end` — End session (D-06 manual end). Response: `SessionResponse`.

- `GET /{session_id}/messages` — Get all messages for session. Response: list of `MessageResponse`.
