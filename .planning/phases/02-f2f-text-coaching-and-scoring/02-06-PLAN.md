**frontend/src/components/admin/scoring-weights.tsx** — Linked sliders totaling 100% per D-08:
- Props: `weights: { key_message: number; objection_handling: number; communication: number; product_knowledge: number; scientific_info: number }`, `onChange: (weights) => void`
- Renders 5 Slider components with labels and percentage display
- Implements `adjustWeights` algorithm per RESEARCH.md Pattern 4:
  When one slider changes, remaining 4 decrease proportionally to maintain total = 100
  `oldValue = weights[changed]`, `diff = newValue - oldValue`, distribute -diff across others proportional to their current values
  Fix rounding errors to ensure sum === 100
- Default: 30/25/20/15/10 (matches backend model defaults)
- Labels from i18n: `admin.scenarios.keyMessageDelivery`, `admin.scenarios.objectionHandling`, etc.
- Each slider shows `{value}%` next to label
- Total display: `100%` (always, as validation ensures it)
