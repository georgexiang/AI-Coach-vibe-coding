1. Create `backend/app/schemas/conference.py` with Pydantic v2 schemas:
   ```python
   from pydantic import BaseModel, ConfigDict, Field

   class AudienceHcpCreate(BaseModel):
       hcp_profile_id: str
       role_in_conference: str = "audience"
       voice_id: str = ""
       sort_order: int = 0

   class AudienceHcpResponse(BaseModel):
       model_config = ConfigDict(from_attributes=True)
       id: str
       scenario_id: str
       hcp_profile_id: str
       role_in_conference: str
       voice_id: str
       sort_order: int
       hcp_name: str = ""
       hcp_specialty: str = ""

   class ConferenceSessionCreate(BaseModel):
       scenario_id: str

   class ConferenceSessionResponse(BaseModel):
       model_config = ConfigDict(from_attributes=True)
       id: str
       user_id: str
       scenario_id: str
       status: str
       session_type: str
       sub_state: str
       presentation_topic: str | None = None
       audience_config: str | None = None
       key_messages_status: str | None = None
       created_at: str | None = None

   class ConferenceMessageSend(BaseModel):
       action: str = Field(description="present or respond")
       message: str
       target_hcp_id: str | None = None

   class QueuedQuestionResponse(BaseModel):
       hcp_profile_id: str
       hcp_name: str
       question: str
       relevance_score: float
       status: str

   class ConferenceSubStateUpdate(BaseModel):
       sub_state: str = Field(pattern="^(presenting|qa)$")
   ```

2. Create `backend/app/services/turn_manager.py`:
   - `QueuedQuestion` dataclass: `hcp_profile_id: str`, `hcp_name: str`, `question: str`, `relevance_score: float`, `queued_at: datetime`, `status: str = "waiting"`
   - `TurnManager` class with methods:
     - `add_question(session_id: str, question: QueuedQuestion) -> None` -- appends and sorts by relevance_score descending
     - `get_queue(session_id: str) -> list[QueuedQuestion]` -- returns waiting questions only
     - `activate_question(session_id: str, hcp_profile_id: str) -> QueuedQuestion | None` -- sets first matching waiting question to "active"
     - `mark_answered(session_id: str, hcp_profile_id: str) -> None` -- sets active question to "answered"
     - `cleanup_session(session_id: str) -> None` -- removes session from _queues dict
     - `get_active_speaker(session_id: str) -> QueuedQuestion | None` -- returns the currently active question
   - Module-level singleton: `turn_manager = TurnManager()`
