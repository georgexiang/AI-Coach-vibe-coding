Create `frontend/src/pages/user/conference-session.tsx`:

Full-screen page (no UserLayout wrapper -- same pattern as training-session.tsx).

1. State management:
   - `sessionId` from useSearchParams `?id=`
   - `useConferenceSession(sessionId)` for session data
   - `useState<TranscriptLine[]>` for transcription lines
   - `useState<QueuedQuestion[]>` for question queue
   - `useState<AudienceHcp[]>` initialized from session.audienceConfig JSON parse
   - `useState<boolean>` for leftCollapsed, rightCollapsed (default: false on desktop)
   - `useState<string>` for streamedText
   - `useState<boolean>` for voiceEnabled
   - `useEndConferenceSession()` mutation
   - `useUpdateSubState()` mutation
   - Speaker color map: `useMemo` building Map<speakerId, colorIndex> from audience config

2. SSE integration:
   - `useConferenceSSE` with callbacks:
     - `onText`: append to streamedText
     - `onSpeakerText`: add TranscriptLine, update audience status for speaker
     - `onQueueUpdate`: set questionQueue, update relevant HCP status to "hand-raised"
     - `onTurnChange`: update audience HCP statuses
     - `onSubState`: show toast, update session sub_state display
     - `onTranscription`: add TranscriptLine
     - `onKeyMessages`: update topic checklist
     - `onDone`: navigate to scoring page or show completion banner
     - `onError`: show error toast

3. Handlers:
   - `handlePresent(message: string)`: call sendMessage("present", message)
   - `handleRespondToQuestion(hcpId: string)`: prompt for response text, call sendMessage("respond", message, hcpId)
   - `handleEndSession()`: show Dialog confirmation (per UI-SPEC destructive action copy), call endConferenceSession mutation
   - `handleVoiceToggle()`: toggle voiceEnabled state

4. Layout (matching UI-SPEC exactly):
   ```tsx
   <div className="flex h-screen w-screen flex-col overflow-hidden bg-background">
     <ConferenceHeader ... />
     <div className="flex flex-1 overflow-hidden">
       <TopicGuide ... />
       <ConferenceStage ... />
       <TranscriptionPanel ... />
     </div>
     <AudiencePanel ... />
     {questionQueue.length > 0 && <QuestionQueue ... />}
   </div>
   ```

5. End session Dialog:
   - Uses Dialog, DialogContent, DialogHeader, DialogTitle, DialogDescription, DialogFooter
   - Title: t("conference.endPresentation")
   - Body: t("conference.endConfirm")
   - Cancel: t("conference.continuePresenting") outline variant
   - Confirm: t("conference.endPresentation") destructive variant

6. Uses `useTranslation("conference")` for all text.
