# Phase 3: Scoring & Assessment Enhancement

## Overview
Enhance the scoring and assessment system with real-time suggestions, post-session reports, and customizable criteria.

### Task 1: Real-time Coaching Suggestions
- Add SSE endpoint for in-session coaching tips
- Display suggestion cards in coaching UI
- AI-driven hint generation based on conversation context

### Task 2: Post-session Report Generation
- Generate comprehensive session reports (strengths, weaknesses, improvement areas)
- PDF/HTML export support
- Historical comparison with previous sessions

### Task 3: Customizable Scoring Rubrics
- Admin UI for creating/editing scoring criteria
- Weight configuration per scenario type
- Dimension templates (communication, product knowledge, objection handling)

### Task 4: Score Analytics Dashboard
- Trend charts for individual MR progress
- Dimension-level breakdown over time
- Benchmark comparison across team
