# Figma Prompt: Scoring & Feedback

Design post-session scoring page showing multi-dimensional evaluation. Desktop 1440x900.

## Top Section:
- Overall score: Large "78" number with circular progress ring (blue), Grade "B+" badge
- Trend: "↑ 5 points vs last session" in green
- Session info: "F2F Training with Dr. Wang Wei — March 24, 2026 — 15 min"

## Two-Column Layout:

### Left Column (50%):
**Radar Chart** (centered, 300x300):
- 5 axes: Key Message (82%), Objection Handling (71%), Communication (85%), Product Knowledge (76%), Scientific Info (68%)
- Current session: solid blue fill
- Previous session: dashed gray line overlay for comparison

**Dimension Breakdown** (below radar, stacked cards):
Each dimension card:
- Dimension name + score bar (colored fill: green>80, orange 60-80, red<60)
- Score number on right

Example:
- Key Message Delivery: ████████░░ 82%
- Objection Handling: ███████░░░ 71%
- Communication Skills: ████████░░ 85%
- Product Knowledge: ███████░░░ 76%
- Scientific Information: ██████░░░░ 68%

### Right Column (50%):
**Detailed Feedback** (scrollable, expandable sections):

For each dimension, a card with:
- Dimension title + score
- ✓ Strengths (green): specific quotes from conversation
  - "Effectively used Phase III survival data to address efficacy concerns"
- ✗ Areas to Improve (orange): specific quotes
  - "Could have addressed safety profile comparison more proactively"
- 💡 Suggestions (blue):
  - "Prepare 2-3 specific statistics for each key message"
