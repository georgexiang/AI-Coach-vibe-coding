## 本手册的来源

本手册内容来源于对 Azure Voice Live + Avatar 功能的深度技术探索，覆盖了从 WebSocket/WebRTC 基础协议到生产环境运维的完整知识链。原始内容以 FAQ 形式记录，现已按学习路径重新组织为渐进式教程。

**适用场景：**
- AI Coach 平台（BeiGene）的 Voice Live + Digital Human 功能开发
- 任何需要 WebSocket + WebRTC 双通道架构的流媒体应用
- Azure AI 服务集成（Speech、Avatar、Realtime API）
