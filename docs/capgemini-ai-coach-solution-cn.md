# 凯捷 AI 教练解决方案

> 凯捷 AI 教练简介 —— 面向医药代表（MR）的 AI 智能培训系统

---

## 1. 解决方案概述

### 面向医药代表的 AI 智能培训 —— 虚拟一对一 HCP 拜访 & 科室会议演练

我们的 AI 驱动数字化 HCP（医疗专业人员）平台，模拟真实医疗场景中的交互体验，具备：

- **拟人化情感深度**，用于异议处理训练及虚拟科室会议演练
- **个性化培训路径**，基于医药代表的职级/业务单元（BU）定制
- **数据驱动洞察**，将练习转化为可量化的成长
- **动态课程优化**，充分发挥 AI 能力

### 核心能力提升方向

- 沟通能力与情境应对能力
- 医学产品演讲技巧

### 两种主要交互模式

| 模式 | 描述 |
|------|------|
| **一对一面谈（F2F Call）** | 与数字化 HCP 进行直接沟通 |
| **一对多会议演讲（Conference Presentation）** | 面向数字化 HCP 听众的虚拟科室学术会议演讲 |

### 技术流程

```
问候语 → 传递核心信息 → 一对一拜访 → 会议演讲 → 异议处理
                                ↓
                              NLP 处理
                        语音处理（语音识别 ASR）
                                ↓
            ┌─────────────────────┼─────────────────────┐
            ↓                     ↓                     ↓
      产品知识库            定制化培训路径           互动提问
                                ↓
                          绩效评估
                                ↓
                        报告与数据看板
```

### 商业价值

| 价值 | 描述 |
|------|------|
| **提升培训效率** | 缩短医药代表上手达标时间 |
| **显著降低成本** | 减少学习与发展（L&D）运营支出 |
| **全程可追溯的培训路径** | 满足合规审计的能力跟踪记录 |

## 2. 核心业务场景

> *数字化赋能高效培训*

### 2.1 培训材料管理

- 集中化文档管理，支持 Word / Excel / PDF / 内容上传
- 培训材料的版本控制与归档
- 按保留策略自动删除语音记录
- 离职员工历史数据归档

### 2.2 一对一 HCP 拜访（F2F 面谈）

- 模拟包含异议处理的面对面拜访场景
- 向医药代表展示场景背景及数字化 HCP 角色信息
- 支持语音和文字输入，数字化 HCP 同步输出对应回应
- 支持历史对话回放与复盘
- 基于评分标准提供评分和反馈
- 提供可自定义的评分标准与反馈体系

### 2.3 虚拟科室学术会议演讲

- 支持含产品/疾病知识的内容演讲，并回答听众提问
- 向医药代表展示演讲的多维场景信息
- 支持语音输入，并在屏幕上实时显示转写文字
- 基于历史数据自动生成典型异议，并提供言语建议
- 提供可自定义的评分标准与反馈体系

### 2.4 报告与数据看板

- 按业务单元（BU）、职级、时间段生成可排序/筛选的标准报告
- 提供个人维度及团队维度的培训结果分析
- 支持 PDF / Excel 格式导出
- 跟踪全组织的培训进度与完成状态

## 3. 用户演示设计 —— HCP 教练

> 展示含数字化 HCP 背景信息的场景，在交互过程中掌握异议处理技巧，无缝切换音频/文字输入以获得即时回应，并通过复盘历史对话持续改进。

### 关键 UI 功能

![HCP 教练 UI 设计](../pdf/images/ui-hcp-coach-demo.png)

#### 3.1 场景与数字化 HCP 背景信息展示
体验真实的医疗场景，结合数字化 HCP 背景信息的可见性，深化对各类医疗情境的理解与准备。

#### 3.2 与数字化 HCP 的异议处理交互
在与数字化 HCP 的交互中进行智能化异议处理，确保沟通清晰高效。

#### 3.3 语音与文字输入及对应输出
支持语音和文字两种输入方式，可接收数字化 HCP 以任一格式作出的准确及时回应。

#### 3.4 历史对话回放
通过回顾和分析历史对话来提升表现，促进持续学习与质量改进。

## 4. 虚拟 HCP —— 多维度评分系统

![多维度评分系统](../pdf/images/ui-scoring-system.png)

多维度评分系统对关键能力进行综合评估，包含：

- **实时建议**：在培训过程中即时提供改进建议
- **课后报告**，重点呈现：
  - 优势
  - 不足
  - 针对性改进方向

### 评分维度

评分系统涵盖以下方面的评估：
- 核心信息传递质量
- 异议处理有效性
- 沟通技巧
- 产品知识准确性
- 科学信息表达能力

## 5. 参考案例：虚拟医生与虚拟教练

![虚拟医生参考案例](../pdf/images/ui-reference-virtual-doctor.png)

### 项目背景
本虚拟 HCP 解决方案为一家顶级跨国企业（MNC）旗下药企打造，帮助医药代表在复杂场景中进行角色扮演练习。借助 AI 驱动的数字人技术，该平台模拟医药代表与虚拟 HCP 之间的真实互动，提供多产品培训、多样化人格设定及语音问答能力，全面提升沟通技巧与产品知识。

### 解决方案亮点

数字人是实现人机类人化交互的创新解决方案。借助人类级别的理解能力和 AI 技术，将销售团队培训提升至新高度：

- **有温度、贴合情境的类人对话**：基于文本分析，帮助识别需求、传递核心信息，推动 HCP 改变观点
- **持续维护与迭代的 NLP 能力**：精准判断和分析情绪、语境、意图等复杂表达，适配不同的 HCP 画像与访谈路径
- **生成深度洞察报告**：评估培训效率，为区域经理（DM）辅导提供建议

### 配置功能

| 功能 | 描述 |
|------|------|
| **设置医生形象** | 自定义虚拟 HCP 的视觉外观 |
| **设置医生知识背景与观点** | 配置领域专业知识和学术立场 |
| **设置评分标准** | 定义评估参数 |

### 商业价值

- **提升专业能力**：改善异议处理技巧和沟通能力
- **优化培训效果与学习效率**：通过个性化、技术赋能的培训加速技能提升，缩短上手时间，加快适应速度
- **降低成本**：通过流程化培训和提升 HCP 互动效率来降低运营成本，最大化投资回报率（ROI）

## 6. 参考案例：虚拟 HCP

![虚拟 HCP 参考案例](../pdf/images/ui-reference-virtual-hcp.png)

### 项目背景
一个角色扮演平台，让医药代表在多样化的医患场景中练习异议处理，AI 驱动的反馈机制帮助精进沟通技巧，改变 HCP 的认知与立场。

### 解决方案亮点

#### 基于场景的仿真演练
提供真实的角色扮演环境，可自定义变量（如患者画像、医生态度），还原真实实地拜访中的挑战。

#### AI 驱动的反馈引擎
多维度评分系统对关键能力进行综合评估，实时提供建议并出具课后报告，重点呈现优势、不足及针对性改进方向。

#### 可扩展与高度可定制
可按特定治疗领域、新产品上市或区域沟通规范灵活配置场景。

### 商业价值

- **提升销售效能**：为医药代表提供经过验证的技巧，提高拜访成功率和产品渗透率
- **数据驱动的绩效优化**：通过聚合分析识别技能差距和培训重点，支持精准辅导和资源配置
- **竞争差异化优势**：可快速响应新兴市场趋势或法规变化，确保长期竞争力
